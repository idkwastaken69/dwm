# dwm - dynamic window manager
dwm is an extremely fast, small, and dynamic window manager for X.

# Installation for newbs
```bash
git clone https://github.com/idkwastaken69/dwm.git
cd dwm
sudo make clean install
```

# Configuration
You can edit the configuration in config.h

# Keybindings
The default keybinds for my build are:
**SUPER+q:** Open terminal (alacritty)
**SUPER+w:** Open web browser (librewolf)
**SUPER+f:** Open file explorer (dolphin)
**SUPER+Space:** Open app launcher (rofi)
**SUPER+c:** Kill window
**SUPER+m:** Exit dwm
**SUPER+Arrow keys/SUPER+Vim motion keys:** Switch between windows
**SUPER+123456789:** Switch tags
**SUPER+Shift+123456789:** Move windows between tags
**SUPER+Minus/SUPER+Equals:** Change window size
**SUOER+Tab:** Toggle between tags
**SUPER+Shift+l:** Lock screen (betterlockscreen)
